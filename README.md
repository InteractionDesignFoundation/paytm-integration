Paytm Integration
=================

1. Create the `.env` file and update the configuration:

    ```
    cp .env.example .env
    ```
2. Install composer dependencies:

    ```
    composer install
    ```
   
3. Run the application using built-in server:

    ```
    php -S localhost:8000
    ```
4. Access the page from browser: `http://localhost:8000`
